**Main Pages**
- Home page
    - Information about what site is
    - Link to sign up/sign in page
    - Accessed from other pages by clicking logo in top left
    - When on this page and not logged in can’t access dashboard, personal timetable, groups pages (profile page labelled as log in)
    - When logged in can access all main pages
- Dashboard
    - Has current day visible
    - Has section about groups
    - Displays next group event if one has been planned
    - Has section about upcoming important events as labelled by user
- Personal timetable
    - Behaves like a usual calendar system
- Groups
    - Hub for each group
    - Has a list of groups to one side
    - Clicking on a specific group will open up the shared timetable for that group
- Profile
    - Contains information about a user’s profile
    - Place where overall settings are managed
