# User Stories

User Stories · CITS3403
Authentication
1 · SP: 2
As a new user, I want to create an account with multi-factor authentication so that I can securely access my timetable and groups from any device.

Merges: create account, MFA on sign up, persistent account across devices


Personal Timetable
2 · SP: 3
As a user, I want to input and update my weekly availability at any time — including blocking out time and setting recurring events — so that my schedule is always current.

Merges: input availability, update availability, block out time, recurring events, persistent calendar on login

3 · SP: 2
As a user, I want to import my university timetable via iCal so that I don't have to re-enter recurring commitments manually.

Merges: import university timetable (×2)


Friends
4 · SP: 2
As a user, I want to find and add friends and be notified when they accept my group invite so that I can build my network and stay informed.

Merges: find and add friends (×2), notification when friend joins group


Dashboard
5 · SP: 3
As a user, I want a dashboard on login showing my upcoming events, current free time, and active group activity so that I can plan my day at a glance.

Merges: dashboard on login, upcoming schedules, quickly see important info

6 · SP: 1
As a user, I want to be reminded of upcoming events so that I don't forget anything important.

Group Management
7 · SP: 4
As a user, I want to create a private group and invite friends via a link or username so that I can organise separate friend circles.

Merges: create or join groups, create private groups, invite via link

8 · SP: 4
As a user, I want to belong to multiple groups and leave or delete a group at any time so that I can manage my participation across different circles.

Merges: belong to multiple groups, delete or leave a group


Group Availability
9 · SP: 5
As a group member, I want to view a shared weekly calendar showing everyone's free and busy times with a visual occupancy indicator so that I can quickly identify the best time to meet.

Merges: shared calendar, view group timetables, visual free/busy indicator, weekly availability view

10 · SP: 4
As a group member, I want to toggle the visibility of my personal events so that I can contribute to the group calendar while keeping private commitments private.
11 · SP: 4
As a member of multiple groups, I want the system to automatically surface the next time each group is collectively free so that I don't have to check each group manually.

Group Event Scheduling
12 · SP: 5
As a group member, I want to suggest a potential event on the group calendar so the whole group can visually see how it fits into their timetables.

Merges: suggest event, schedule events for the whole group

13 · SP: 5
As a group member, I want an agreed event to be automatically inserted into each member's personal timetable so that everyone's availability updates across all groups.
14 · SP: 3
As a group member, I want to leave notes on the group calendar so that I can suggest ideas and discuss plans with the group.
