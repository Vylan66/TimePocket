1. As an user, I want to be able to create private groups with specific users, so that I can separate different friend groups
2. As a group member, I want to be able to see everyones timetables, so that I can see when the whole group is collectively free
3. As a University student, I want to be able to upload my university timetable, so that I don't have to manually manage two timetables
4. As a member of multiple groups, I want the system to automatically show when the next time each group will collectively be free, so that I don't have to check and remember each individual group's availability. 
5. As a group member, I want to be able to leave notes for the group to see, so that I can suggest things to do
6. As a group member, I want to be able to suggest a potential event on the group timetable, so that the whole group can visually see how that event will fit into their timetable
7. As a group member, I want the system to automatically insert an event into each group members personal timetable once a suggested event has been agreed upon, so it will update my availability across all groups
8. As a user, I want the system to filter important details from my personal timetable, so that my groups will only see necessary details about my events to protect my privacy
9. As a group member, I want the system to suggest events to do tailored to each group and its member's preferences, so that we have an event to do in case we can't decide what to do.
10. As a group member, I want each timetable to be visually simple and easy to understand, so I can quickly tell when each person is free.
