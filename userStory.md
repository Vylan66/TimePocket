# User Stories

| # | Backlog Item (User Story) | Story Points |
|---|---|---|
| 1 | As a user, I want to create an account so that I can access my personal timetable and groups. | 2 |
| 2 | As a user, I want to create a group and invite friends via a link so that we can share our availability. | 3 |
| 3 | As a user, I want to input my weekly availability so that my group can see when I am free. | 3 |
| 4 | As a user, I want to view a dashboard showing which of my friend groups are currently free so that I can plan catch-ups quickly. | 4 |
| 5 | As a user, I want to view a weekly calendar of my group's shared availability so that I can find common free times. | 4 |
| 6 | As a user, I want to receive a notification when a friend joins my group so that I know my invite was accepted. | 1 |
| 7 | As a user, I want to update my availability at any time so that my group always sees my current schedule. | 2 |
| 8 | As a user, I want to be able to belong to multiple groups so that I can manage availability across different friend circles. | 2 |
| 9 | As a user, I want to see a visual indicator of how many group members are free at a given time so that I can quickly identify the best meeting times. | 3 |
| 10 | As a user, I want to delete or leave a group so that I can manage which groups I am part of. | 1 |
