1. As an unregistered user, I want to have Multi-Factor Authentication (MFA) upon sign up, so I can ensure my credentials are safely stored, managed, and accessed.
2. As a registered user, I want my calendar data to be persistent upon each update, so I can avoid updating the calendar each time upon log-in.
3. As a user, I want to be able to update my personal calendar with my personal schedule, so I have a good time management and productivity system.
4. As a user, I want to be able to search for friends and send friend requests on the app, so I can form or invite trusted connections.
5. As a user, I want to be able to form connections with friends, so I can create or join groups.
6. As a connected user, I want to be able to create or join groups, so we have a shared calendar with compiled schedules.
7. As a group user, I want to have a shared calendar among the group, so we can easily identify common free time.
8. As a group user, I want to be able to toggle my personal activity's visibility, so I can contribute to the group calendar while preserving my privacy if I wish to.
9. As a user, I want to have the ability to be part of multiple groups, so I can coordinate my free time with different social circles for hangouts.
10. As a user, I want to have a dashboard page on the app upon log-in, so I can conveniently see my upcoming schedules, free time, or group activities.
